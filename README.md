# First Synthesis

First Synthesis LLC develops precise mathematical instruments. Our primary technology, the **Monist Engine**, pushes the boundaries of unstratified logic, combinatorial reduction, and formal verification against modern hardware limits.

## The Monist Engine

The Monist Engine is a deterministic execution layer designed exclusively to compute Quine's New Foundations (NF) unstratified, self-referential logic. By abandoning traditional hierarchical type-checking in favor of geometric shortest-path routing, the project compiles structurally dense paradoxes into native, lock-free Interaction Nets.

### Key Capabilities

* **Autonomous GPU Dispatch**: Executes highly parallel combinatorial reductions directly in VRAM, bypassing CPU synchronization overhead using a 32-bit tagged pointer system.
* **The Oracle Layer**: Transforms standard first-order logic into algebraic constraint matrices and synthesizes dynamic T-Functors to absorb structural friction.
* **Formal Verification**: Continuously compiles structural topologies into SMT-LIB v2 witnesses directly into Lean 4 for parallel proof-theoretic validation.
